## 역할 분리

- GitHub Pages: 공개 원본, 검색 노출, 정적 산출물 저장의 본진. 기본 URL은 `https://r2cuerdame.github.io/radar/`.
- Tistory/외부 블로그: 2026-04-29 의종 결정으로 폐기. dedicated publisher, browser fill, manual package, 로그인 확인 재시도 모두 중단.
- #naver 보고 채널: 중요한 blocker/milestone만 받는 목적. 원고 전문과 장문 로그 노출 금지.

## 막힘 처리

- GitHub Pages 빌드/검색 반영 지연은 `pages_publish_blocked` 또는 `indexing_pending`으로 기록한다.
- 로그인, 캡차, 권한, 카테고리, 스킨, 광고 설정 문제는 Tistory 쪽이면 더 이상 해결 대상으로 잡지 않는다.
- 해당 글은 삭제하지 않고 `published_revision`, `release_candidate`, `publish_held`로 keep한다.
- 다음 후보 제작, 자료 확인, 편집/시각자료 보강은 계속한다.

## 공개 본문 경계

- 현재 전략은 free-complete/public magazine이다.
- 공개 글은 완성형이어야 하며, 독자를 일부러 끊어 세우는 문구를 쓰지 않는다.
- 직접 판매, 계정 변경, 제휴형 구매 유도, 내부 stage 명칭, Tistory 복붙/보안확인 문맥은 동네 레이더 공개 본문에서 제외한다.
