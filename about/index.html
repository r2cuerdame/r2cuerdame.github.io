무엇을 올리나

동네 레이더의 지역 데이터 글, 생활 가이드, 검수된 파트너스 추천글을 분리된 섹션에 공개합니다.

현재 공개 수

파트너스 픽 12개, 동네 레이더 1개를 기준으로 sitemap/RSS에 반영합니다.

왜 GitHub Pages인가

새 도메인 없이 빠르게 열 수 있고, 원본 파일/배포 이력이 GitHub에 남으며, SEO 기본 파일을 직접 제어할 수 있기 때문입니다.
