# 복도 끝 집, 조용함인지 사각지대인지 보는 밤 12분 — 30-story

- 상태: `story_pass`
- 흐름: 첫 화면 독자/결정상황 → 복도 끝의 장점과 착시 → 밤 귀가 시야 → 문 앞 머무름 → 조명·계단실 반례 → 흐린 답변 질문 → 12분 사용법 → 비슷한 공용부 후보 분리 → 다음 확인 하나 선택 → 관리·중개 답변 재질문 → 동거인/가족에게 감정이 아니라 확인 장면으로 설명.
- 핵심 해석: 조용함은 좋은 신호가 될 수 있지만, 시야가 끊기고 사람 흐름이 설명되지 않으면 고립감으로 바뀐다.
- 반례: 복도 끝이어도 조명, 완충 폭, 비상구 방향, 이웃 출입이 분리되면 계속 볼 이유가 남는다.
- 독자 문장: “복도 끝이라서 조용하다”를 “내 귀가 시간에 시야·소리·도움 요청 동선이 분리되는가”로 바꿔 말한다.
- 2026-05-07T08:38:48+09:00 보강: 마지막 행동 언어를 `조용해서 좋다/무섭다`가 아니라 `밤 시간·센서등·방 안 소리·문 앞 머무름`으로 설명하게 만들어 같이 결정하는 독자에게도 비교 기준이 남도록 했다.
